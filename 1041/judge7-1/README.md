# NCNU Programming 1041 Judge 7-1

大家的學姊的老師的程式設計課的 TA 的 TA 的學妹小青最近好像剛上幼稚園，最近剛開始學數數字，小青從 1 開始數，天資聰穎的正在念幼稚園的小青突發奇想的想知道剛剛自己數的數字中有幾個 5，然後還想把有 5 的數字都列出來，請各位同學幫幫大家的學姊的老師的程式設計課的 TA 的 TA 的學妹小青吧！

## 題目

1 到某個數之中包含多少個 5

## 輸入

要數到多少

## 輸出

- 列出所有結果
- 有幾個這種數字
- 包含了多少個 5
