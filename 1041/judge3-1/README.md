# NCNU Programming 1041 Judge 3-1

因為最近姜姜非常缺錢,所以決定放手賭一把跑去買了他好姊妹小乖乖所經營彩卷行的彩卷,希望重大獎就可以不停地揮霍的買專輯買鞋都買都買。但是彩券行的開獎機壞了,希望各位同學幫小乖乖用程式弄出一台新的開獎機來!!

## 題目

會有一個頭獎號碼和其他五個不重複的號碼。首獎必須為質數，結果請按照降冪排列。(由1到55號)

## 輸入

無

## 輸出

```
首獎: 03
其他: 54 36 32 29 05
```

```
首獎: 23
其他: 50 40 29 26 04
```

```
首獎: 31
其他: 40 20 08 06 05
```
