# NCNU Programming 1031 Judge 6-2

在程設上機考的時候，老俞看到同學們都寫不出第一題，決定出手出第二題。要大家幫他寫一個十轉 n 進位的程式。
 
## 輸入

兩整數 val, base

## 輸出

將 val 以 base 進位系統印出，若 base > 10，則依序以大寫英文表示。

## 提示

 ‘A’ = 65, ‘B’= 66, ‘C’= 67...

## Sample input

```
10 2
```

```
15 16
```

```
10 11
```

```
123 16
```

## Sample output

```
1010
```

```
F
```

```
A
```

```
7B
```
